# Опис набору даних

## Назва набору даних
Реєстраційні операції з транспортними засобами в Україні

## Загальна характеристика

Набір даних містить інформацію про реєстраційні операції з транспортними засобами в Україні за період з 1 січня 2022 року по 31 грудня 2022 року.

Розмір даних:
- 1 745 908 записів
- 20 колонок
- Формат: CSV
- Обсяг файлу: ~603 MB

## Опис основних полів

| Поле | Опис |
|------|------|
| PERSON | Тип власника (P – фізична особа, J – юридична особа) |
| REG_ADDR_KOATUU | Код адміністративної території |
| OPER_CODE | Код операції |
| OPER_NAME | Назва реєстраційної операції |
| D_REG | Дата реєстрації |
| DEP_CODE | Код сервісного центру |
| DEP | Назва сервісного центру |
| BRAND | Марка транспортного засобу |
| MODEL | Модель |
| VIN | Ідентифікаційний номер транспортного засобу |
| MAKE_YEAR | Рік випуску |
| COLOR | Колір |
| KIND | Тип ТЗ (легковий, вантажний тощо) |
| BODY | Тип кузова |
| PURPOSE | Призначення |
| FUEL | Тип пального |
| CAPACITY | Обʼєм двигуна |
| OWN_WEIGHT | Власна маса |
| TOTAL_WEIGHT | Повна маса |
| N_REG_NEW | Новий державний номерний знак |
